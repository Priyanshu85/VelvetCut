## üöÄ Overview

**VelvetCut** is a browser-based video editing suite designed for speed and simplicity. It allows creators to trim, crop, filter, and master video content entirely client-side, ensuring privacy and zero-latency performance.

## ‚ú® Features

- **Client-Side Processing:** No uploads required. Your videos never leave your device.
- **Smart Trimming:** Precise timeline controls for cutting and arranging clips.
- **Visual Filters:** Cinematic presets (Noir, Vintage, Cinematic) powered by CSS filters.
- **Composition Tools:** Presets for Social Square (1:1), Reels/Shorts (9:16), and more.
- **Playback Control:** Variable playback speeds (0.5x - 2x) for detailed review.
- **Privacy First:** Built with privacy in mind; no external API keys needed.

## ‚ö° Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Priyanshu85/VelvetCut.git
   cd VelvetCut
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   npm run preview
   ```
